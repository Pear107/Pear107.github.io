常见错误

这五个闭包在循环中被创建，但他们共享了同一个词法作用域，在这个作用域中存在一个变量 i。这是因为变量 i 使用 var 进行声明，由于变量提升，所以具有函数作用域。当数组赋值时，i 的值被决定。由于循环在数组赋值之前早已执行完毕，变量对象 i（被五个闭包所共享）已经指向了 i 的最后一项。
